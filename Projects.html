Sistema de Gestión de Órdenes

Aplicación full stack desarrollada para optimizar la eficiencia operativa en la gestión de órdenes. Se implementó una API RESTful con C#, ASP.NET Core y Entity Framework Core bajo una arquitectura en capas, integrada con Oracle Database para la persistencia de datos transaccionales. Se creó además una aplicación móvil multiplataforma con Flutter (Dart) que consumía estos servicios. Se aplicaron principios SOLID, prácticas de Clean Code y pruebas automatizadas para garantizar mantenibilidad y calidad del software.
